# haome525 导航站

个人技术工具导航 — AI · 开发 · 效率精选

## 特性

- 50+ 精选工具，8 大分类
- 多引擎搜索（站内/Bing/Google）
- 分类筛选 + 关键词过滤
- 三主题（纸墨/深空/石墨）× 明暗模式
- 响应式设计，移动端友好
- 纯静态，零依赖，秒开
- 快捷键 `/` 聚焦搜索

## 本地预览

```bash
# 使用 Python
python -m http.server 8080

# 或使用 Node
npx serve .
```

浏览器打开 http://localhost:8080

## 数据维护

编辑 `data.json` 即可新增/修改/删除导航站点，无需改动 HTML/CSS/JS。
